Installation:

  • 1. Entpacke das komplette Archiv und speicher es auf deiner Festplatte ab ( per winrar )
  • 2. Uploade dieses Theme in den Theme Ordner auf deinen Server ( per filezilla )
  • 2. Uploade die nötigen Plugins in den Plugin Ordner auf deinen Server
  • 4. Geh in den Administrationsbereich und klicke auf Design
  • 5. Aktiviere das Infinity WordPress Theme
  • 6. Nach aktivieren der Plugins klicke auf "Einstellungen" um flickrRSS zu konfigurieren.
  • 7. Das war's schon Viel Spaß beim Bloggen.

Thumbnails, Delicious und Twitter Integration

Wie du wahrscheinlich schon gesehen hast benötigt das Infinity Theme ein benutzerdefiniertes Feld damit das "Magazine cover style" funktioniert, weil es sich doch sehr von den 'Standardeinträgen' abhebt.
Der clevere Trick ist folgender: Bevor du deinen Post veröffentlichst scrolle etwas herunter zu den Benutzerdefinierten Feldern:

Name: thumbnail
Wert: Pfad des Images

Sobald du den Image Pfad eingegeben hast klicke auf Publizieren und deine letzten Beiträge können als Thumbnail Image (235*150 als Standardwert) angezeigt werden.
Willst du dieses Image später ändern gehe einfach auf Artikel bearbeiten, scrolle nach unten bis zu den benutzerdefinierten Feldern und ändere den Link neben thumbnails und klicke anschließend auf Aktualisieren, und das Image wird geändert.

Twitter-Zeug
Öffne die Datei "left.php" und geh in die Zeile 8 und ersetze den Bildlink durch deinen eigenen.

<script type="text/javascript" src="http://twitter.com/statuses/user_timeline/yichi.json?callback=twitterCallback2&count=1"></script>

Feine Bookmarks
Öffne die Datei "left.php", geh in die Zeile 25 und ersetze die ID (hier: yichi) mit deiner eigenen:

<script type="text/javascript" src="http://feeds.delicious.com/v2/js/yichi?count=8&sort=date"></script>

Das ist alles, das Theme wird super funktionieren. Wenn du irgendwelche Fragen hast oder deine Meinung über das Theme abgeben willst freue ich mich schon darauf dir helfen zu können.

Thumb Image

  • 1. Finde ein angebrachtes Image, ändere die Größe (max 320px width) und lade es im Administrationsbereich hoch.
  • 2. Kopiere die Image-URL
  • 3. Scrolle im Post Admin Panel zu den benutzerdefinierten Feldern, und wähle Name aus
  • 4. Füge "thumbnail" ins Namenfeld ein
  • 5. Füge die kopierte URL in den Wert ein
  • (Beispiel:)http://domain.de/uploads/2007/09/vj.gif
  • 6. speichern
  • 7. Probiere eine Vorschau aus. Wenn alles glatt läuft klicke auf "Publizieren".
