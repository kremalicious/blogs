Colortree.zip file in a convenient location	* Copy the 'Colortree' folder to '/wp-content/themes/' directory in your web server.	* Log in to the WordPress Administration Panels	* Select the 'Themes' sub-menu from 'Appearance' menu (or 'Design'/'Presentation' depending on the WP version you are using).	* Select the Colortree theme from the available list of themes to preview	* Activate the theme.	* Your selection should now become active.License------------------------------------------------------------------------		"Colortree" is a WordPress theme	Copyright
