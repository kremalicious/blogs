WP FollowMe plugin allows you to add a twitter "Follow me" badge on your wordpress blog.

== Description ==

[WP FollowMe](http://wpburn.com/wordpress-plugins/wp-followme-plugin) is a wordpress plugin that allow you to add a twitter "Follow me" badge on your wordpress blog.

= Update in 2.0.8 =
* Fixed display in some themes. Thanks to [Austin](http://wordpress.org/support/topic/wp-followme-simply-wont-display-thesis-theme?replies=5)!

= Update in 2.0.3 =
* The badge is completly in Flash.
* The text is displayed sideways.
* Added border color option.


= With FollowMe you can =

* add a FollowMe Badge on your wordpress blog
* change the color of the badge
* change the follow me label
* change the position of the badge (left side or right side and the distance from top)

= How to use it =

* Download it
* Extract all files from the zip archive to your wp-content/plugins folder
* Activate the plugin through the "Plugins" menu in WordPress
* Add your Twitter URL and configure other options as you want
* That's it!

You can see a demo on my website: [WPburn.com](http://wpburn.com)
