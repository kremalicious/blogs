	Installation
	------------

	Unzip the file '*.rar' and upload the complete content to:
	
	/wordpress_root_dir/wp-content/themes/

	In your WordPress Admin Panel go to 'Themes' and activate the theme.

	==============================================================================

	Entpacke die Datei '*.rar' und lade den gesamten Inhalt in das Verzeichnis:

	/wordpress_root_dir/wp-content/themes/

	Aktiviere das Template im Wordpress Admin-Bereich.
