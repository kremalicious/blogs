The Events Calendar Widget that displays the events filtered by category. Depends on "The Events Calendar" plugin.

== Description ==

Use this plugin to filter the events by the category. 



== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `The Events Calendar Widget` directory to the `/wp-content/plugins/` directory
2. Install The Events Calendar plugin and activate it.
3. Then activate the `The Events Calendar Widget` plugin through the 'Plugins' menu in WordPress
