== Description ==
= wpMAPS bringt Google Maps und WordPress zusammen =
Durch die Setzung des Shortcode (= Platzhalter) im Artikel ist das wpMAPS-Plugin in der Lage, Google Maps direkt im Beitrag einzubinden und darzustellen - mehrfach pro Blogseite und mit jeweils abweichenden Darstellungsoptionen.
Auch aus dem Template heraus kann die Ausgabe der Karten gesteuert werden. Dafür steht eine interne PHP-Funktion zur Verfügung, die ebenfalls mit Parametern ausgestattet werden kann.

Besuchen Sie die [wpMAPS Beschreibungsseite](http://playground.ebiene.de/1379/wpmaps-wordpress-google-maps/ "wpMAPS Beschreibungsseite") für weitere Informationen und ausführliche Beispiele.
